

------PROPOSTA----------

Um sistema bancário simples desenvolvido por Joana Silva em Java que permite gerenciar contas bancárias com operações básicas.

------FUNCIONAMENTO-----

Este projeto implementa uma simulação de sistema bancário onde os usuários podem:
-Criar uma conta bancária
-Realizar operações financeiras básicas
-Visualizar informações da conta
-Gerenciar o status da conta
-Apagar a conta

------PRÉ-REQUISITOS----:

-IDE

------EXECUÇÃO----------

-Dê um fork e um clone no repósitorio 
-Execute o programa main.java
